# Duel Sprites

Sprite assets for the duel (card battle) screen. All files use the `.pic.png` format.

## Sprite Categories

### Grave_* - Graveyard
Empty graveyard images, one per color: Black, Blue, Green, Red, White.

### Hand_* - Player Hand
Container image to hold cards in a player's hand, one per color: Black, Blue, Green, Red, White.

### Terr_* - Player Board Backgrounds
Board terrain backgrounds, three variants per color (mana, patt, pict): Black, Blue, Green, Red, White.

### Winbk_* - Window Backgrounds
UI panel backgrounds for various duel screens:
- **Manapool** - Player's mana pool background
- **Phase / Phasecombat** - List of player's phases
- **Spellchain / Spellmin** - Current stack of spells/abilities waiting to be processed
- **Attack / Attackbones / Attackmin / Attackrats / Attackshield / Attacksword** - Combat UI elements
- **Ante / Antelabel** - Ante selection screen
- **Bigcard** - Enlarged card view
- **Changetext** - Text change dialog
- **Credits** - Credits screen
- **Endduel** - End of duel results
- **Fireball** - Fireball X cost prompt
- **Foilpackhighlights** - Foil pack highlight overlays
- **Ladder** - Tournament ladder
- **Manaburn** - Mana burn warning
- **Options** - Options menu
- **Questmana / Questmanaselection** - Quest mana selection
- **Questn** - Quest dialog
- **Rogue** - Rogue encounter
- **Shellspherehighlight** - Shell of the Sphere highlight
- **Startduel / Startduel2 / Startduelbutton*** - Duel start screen and buttons (normal, depressed, disabled)
- **Telluser** - User message/notification box
- **Versus** - Player vs opponent splash

### Abilities
Sprite sheet of ability/keyword badges to overlay on cards (flying, deathtouch, trample, etc).

### Cardcounters
Sprite sheet of counters that can be added to cards (+1/+1, haste, etc).

### Other
- **Damage** - Damage indicator
- **Summon** - Summoning sickness indicator
- **Target** - Targeting cursor/indicator
- **Canttarget** - "Can't be targeted" indicator
