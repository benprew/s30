# Shandalar 30th Anniversay Edition 0.4.0 Release Notes

Hey Y'all back with another monthly update on [Shandalar 30th Anniversary](https://github.com/benprew/s30/releases/tag/v0.4.0). I got my first bug report! Feydan submitted a report about how attacking creatures weren't clear in the UI so I added a little animation to clarify when a creature is selected to attack.

I appreciate everyone's feedback on the game and please keep sending them in, thanks!

Lots of little improvements this month, it's starting to feel more like a game instead of a tech demo. Now when I go to test something, I find myself actually playing through the game more. There's still a lot of work to do, but it feels good.

## Highlights

- New game screens, with difficulty and color selection!
- Saves are grouped by play-through instead of showing all auto saves
- [beta] Wizard Castles are available! When you go to a castle it shows you the duel screen for the wizard. I'm still working on adding a dungeon to the castles.
- Random enemies scale better - before any random enemy could show up, now it's based on time, the longer you play the harder the enemies get
- Mulligan - uses London Mulligan rules not the old system which was only mulligan if no land or if opp. mulligans
- AI is getting better, understands combat better now, still struggles with some bonehead moves
- Card engine supports Jumpstart and Secrets of Strixhaven, but they're not available in the game yet
